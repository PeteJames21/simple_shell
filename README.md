# Simple Shell

## Description
A simple UNIX command interpreter.

## Usage
The project is compilable on Ubuntu >= 20.04 using the following command:
```
gcc -Wall -Wextra -Werror -pedantic -std=gnu89 -Wno-format *.c
```
